• 现在比较像“热度榜”的入口主要有这几个：

  - AgentSkillsRepo (https://agentskillsrepo.com/)：有 Top Skills、分类、搜索结果里也会显示安装量和趋势，比较像公开排行榜。
  - AgentSkills.to (https://agentskills.to/)：首页直接有 Most popular this week，适合看近期热门。
  - Skill Marketplace (https://skillmarketplace.ai/)：支持按 popularity、最近更新、相关性排序。
  - Learn Skills (https://www.learn-skills.dev/)：有 Trending 和 Newest 两种榜单视图。
  - SkillsLLM (https://skillsllm.com/)：首页有 Trending Skills，偏聚合展示。

  如果你是想找 Codex 可用的 skill，优先看：

  - AgentSkillsRepo (https://agentskillsrepo.com/)
  - Skill Marketplace (https://skillmarketplace.ai/)
  - Learn Skills (https://www.learn-skills.dev/)
