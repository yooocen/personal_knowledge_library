# 平台介绍（小艺开放平台）图片学习总结

更新时间：2026-02-09 13:49（来自截图）

## 1. 平台定位
- 小艺开放平台依托鸿蒙系统生态，为开发者提供全链路的智能体开发解决方案。
- 提供端到端工具链，覆盖智能体开发、多端调试（手机/平板/车机/PC/手表）到部署上架的全生命周期。

## 2. 四大核心开发模式
- LLM 模式：大语言模型驱动。
- 工作流模式：可视化流程编排。
- A2A 模式：直连三方智能体。
- OpenClaw 模式：个性化助手。

## 3. 上架与分发
- 智能体开发完成后，统一上架至智能体市场，便于集中管理，并面向多设备、多入口分发。
- 典型流程：设计/开发/测试 -> 上架/分发/运行。

## 4. 三类触达入口
1. 鸿蒙系统级入口：通过系统导航条、小艺搜索、小艺建议等系统入口快速触达相关智能体。
2. 小艺入口：小艺作为系统中的超级智能体，可通过复制、拖拽、圈选等交互，结合屏幕内容与用户意图进行任务规划拆解，并协同多个智能体完成任务。
3. 应用内入口：用统一服务标识和上下文感知的轻量入口，让智能体与应用核心体验无缝融合，从“需要主动发现/调用的独立功能”转为“按需涌现的自然流动服务”。
