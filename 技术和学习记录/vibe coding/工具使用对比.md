# 安装上手

| 工具          | 安装方式                                                                                                                              | 评价                                                         |
| ----------- | --------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------- |
| Claude Code | 1. 准备环境：安装nodejs 18+<br>2. 执行命令 `curl -fsSL https://claude.ai/install.sh \| bash`<br>3. 登录anthropic账户                             | 如果有Anthropic的账号，几步就可以安装好了。<br>如果想用国产模型，可以参考智普官网的教程。        |
| OpenCode    | 1. 准备环境：安装nodejs 18+<br>2. 执行命令 <br>`curl -fsSL https://opencode.ai/install \| bash~/.bashrc`                                     | 安装简单，上手快<br>开箱即用，OpenCode内置有免费模型。<br>如果想用国产模型，可以参考智普官网的教程。 |
| Codex       | 1. 准备环境：安装nodejs 22+<br>2. 安装：`npm install -g @openai/codex`<br>3. 首次登录：运行 `codex`（或 `codex login`），按提示用 ChatGPT 账号或 API Key 完成授权 | 安装简单，有账户也是贼方便。如果想便宜，可以在淘宝买一些api key                        |
|             |                                                                                                                                   |                                                            |

# 总结
个人用下来，目前是Codex、OpenCode会使用更多一点，Claude Code主要是价格和国内环境的问题用的不多。vibe coding确实比以前的手写代码提升效率，主要体现在AI对于工作流的帮助是全流程的，以前可能只是写写代码片段之类的，现在AI工具可以帮你克隆代码，帮你分析，直接帮你改bug等等，对于程序员来说就会是更多是专注于自己想要实现什么功能，设计什么特性，不需要关心环境问题，细节问题等一系列可能也重要但是有点浪费时间的事情，提升自己的专注度的同时，有效工作时间会增加（以前为了解决一些网络不通的问题，可能直接从开始到放弃，现在只要给AI一点时间，它会自己解决，甚至还可以从它的解决方式上学到一些技巧）
