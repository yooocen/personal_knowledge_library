# Claude Code

基于 Claude 系列模型，当前常见能力组合是 Opus / Sonnet / Haiku  3个挡位
目前主流是Claude 4模型

- **Claude Opus 4.6 (旗舰版)**
    - **定位**：最高智能水平，适用于科研、金融分析、复杂代码库重构等极端复杂任务。
    - **能力**：支持“自适应思考”和“智能体团队”模式，可自动拆分并执行多步骤任务。
    - **上下文**：默认20万 token，测试版支持100万 token。
    - **API 定价**：输入约 $5/百万 token，输出约 $25/百万 token。
    
- **Claude Sonnet 4.6 (平衡版)**
    - **定位**：性能接近 Opus，但价格仅为五分之一，是性价比最高的选择，也是多数用户的默认模型。
    - **能力**：在编程、长上下文推理和智能体规划方面表现突出，支持100万 token 上下文（测试版）。
    - **API 定价**：输入 $3/百万 token，输出 $15/百万 token。
    
- **Claude Haiku 4.5 (轻量版)**
    - **定位**：速度最快、成本最低，适用于高并发、低延迟场景，如实时对话和简单分类任务

注意：Claude Code最好搭配用它自己的闭源模型效果最好，缺点就是太贵了。它也可以改为国产模型，比如智普的glm系列，minimax等，但是需要自己改配置文件，或者使用Claude Code Router这种专门的格式转换的开源软件来使用，相对来说比较麻烦

个人评价：好用是好用，对国内用户不友好是真不友好
# OpenCode

本质是开源 AI 编码代理，不绑定单一模型提供商。支持多模型、多供应商和本地模型，强调“按场景切模型”。
OpenCode目前支持接入很多厂商的大模型（列一些我尝试过，比较好用的）
- Claude （opus 、sonnet）
- OpenAI  (gpt-5.x)
- Google (gemini 3)
- Z.AI （glm 4.7）
- deepseek
- alibaba（qwen）
- Moonshot ai（kimi）
- MiniMax（M2.5）

OpenCode还有一个Zen网关，相当于买它一个统一的套餐，里面就已经有一堆厂商的模型可以用，不需要自己单独去订阅厂商的套餐

个人评价：切换模型很方便，推荐一下智普的glm 系列，一个月大概50块，输出快，效果也不错（好像 0.3🔪/百万Token）。对国内用户使用是最方便的了。

# codex

OpenAI出品，也是用的自家的闭源模型gpt系列效果最好，主要有下面几种
- gpt-5.2
- gpt-5.2-codex
- gpt-5.1-codex-max
- gpt-5.1-codex-mini
带codex后缀的就是为Codex优化过的模型，不带的就是ChatGPT用的模型，效果也是从夯到拉

个人评价：codex的模型效果感觉比肩Anthropic，而且价格便宜的同时，对国内用户相对友好
