## Highlights
- 随着业界前后端分离的趋势，以及前端应用表现力和交互能力的增强，越来越多的逻辑承载在了前端，追求原生应用体验的SPA（Single-Page Application）在新窗口打开前端架构模式成为了大部分现代企业应用的选择。
- UI 模型 概括来说，包含数据和行为两部分。其中：数据 指的就是上文说的前端状态（UI State），以及前端状态的结构描述（Schema of UI State）行为 指的就是围绕数据的动作行为、逻辑运算，例如发送后端请求、根据业务逻辑计算前端状态、进行数据转换处理等
