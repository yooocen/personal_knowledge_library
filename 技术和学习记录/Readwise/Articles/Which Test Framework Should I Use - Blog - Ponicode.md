## Highlights
- These are libraries or tools that launch and execute your tests and aggregate their results.
- Jasmine is an open source testing framework that comes ready out-of-the box, without a need for additional assertion libraries nor mocking libraries (although more libraries can be added if specific features are required). The fact it has no external dependencies means it’s relatively fast.
- Jasmine also has its own inbuilt test runner. It can run tests both in the browser and directly in Node.js, from the command line, and does not require the DOM. This makes it a popular choice for web apps. As a matter of fact, Jasmine was created with the aim of being browser, framework, platform and language independent, and it can be used not only with JavaScript but also with Ruby and Python.
- Mocha is the second most popular test framework for JavaScript, and it is open source.Differently from Jest and Jasmine, Mocha is only a test framework: it has no test runner, no built-in assertion libraries and no default mocking tools, therefore it requires its users to choose third-party plugins depending on their project and their testing needs. The most popular choice of assertion library to pair with Mocha is Chai, but other options are Should.js, Express.js, Unexpected and Better-assert.
